## 📌 Project Overview

**VibeNest** is a modern and responsive e-commerce landing page that introduces a new way of shopping - based on **user mood**.

Instead of browsing endless products, users explore items through emotions like:

- 😊 Happy  
- 🤗 Comfort  
- 🧭 Adventure  
- 💕 Romantic  

The goal is to create a shopping experience that feels **simple, personal, and human-centered**.

## 🚀 Key Features

### 🎬 Welcome Experience
- Animated entry screen  
- Floating emojis & glowing UI  

### 🧭 Navigation & Header
- Clean and minimal navbar  
- Smooth scrolling between sections  

### 🎯 Hero Section
- Eye-catching headline: *"Shop For The Mood You're In"*  
- Mood-based call-to-action  

### ⚙️ How It Works
1. Select your mood  
2. Get curated products  
3. Shop & feel good

- Clean and modern user feedback UI

- Brand information
